# Signature-Based IPS

A Signature-Based Intrusion Prevention System built using Python and Scapy.

## Lab Setup

IPS Engine: 192.168.56.1
Victim Server: 192.168.56.2
Kali Attacker: 192.168.56.3

## Technologies

- Python
- Scapy
- VirtualBox
- Ubuntu
- Kali Linux
  # Setup

1. Clone the repository:
   git clone https://github.com/Ameer8206/signature-based-ips.git

2. Install dependencies:
   pip3 install scapy

3. Run the IPS engine:
   python3 gui.py

## Requirements
- Python 3.x
- Ubuntu 22.04 LTS
- Scapy
